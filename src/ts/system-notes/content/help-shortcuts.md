# Keyboard Shortcuts

## Global

| Key | Action |
|-----|--------|
| `Ctrl+S` | Save current note |
| `Ctrl+N` | New note |
| `Ctrl+E` | Toggle edit mode |
| `Ctrl+F` | Focus search |
| `Ctrl+B` | Toggle sidebar |
| `Backspace` | Go back (when not editing) |

## Editor (CodeMirror / Raw)

| Key | Action |
|-----|--------|
| `Tab` | Indent |
| `Shift+Tab` | Outdent |
| `Ctrl+/` | Toggle comment |
| `Ctrl+Z` | Undo |
| `Ctrl+Y` | Redo |
