# Emoji

The emoji plugin adds support for typing emoji via shortcodes.

## Usage

```markdown
I :heart: Leaf   →   I ❤️ Leaf
```

Wrap the emoji name in colons — `:rocket:`, `:tada:`, `:smile:`.
